Penggunaan

Instalasi

· Masukkan JQuery v2.2.4


· Masukkan file JavaScript City Selector


≫. Pastikan Folder "city" tidak berada dalam folder lain. Disarankan agar folder "city" ditempatkan pada folder utama project anda.

Pembuatan

· Buat 2 Elemen Select


Element Select yang pertama akan menampilkan list Provinsi.
Sedangkan Elemen Select kedua akan menampilkan list Kabupaten/Kota.

Agar City Selector dapat berfungsi, berikan Class city-init pada Elemen Select Provinsi.

<select class="city-init"></select>
Berikan atribut city-target dan berikan nilai dari ID Elemen Select Kabupaten/Kota.
<!-- Provinsi --> <select class="city-init" city-target='#targetkabupaten'></select> <!-- Kabupaten/Kota --> <select id="targetkabupaten"></select>
